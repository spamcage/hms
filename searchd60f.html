主题: roundcube webmail配置后好无草稿箱，已发送邮件箱、废件箱、垃圾箱。解决方法。
回复总数: 0
阅读次数: 16780

roundcube webmail配置后好无草稿箱，已发送邮件箱、废件箱、垃圾箱。解决方法。

roundcube webmail配置后好无草稿箱，已发送邮件箱、废件箱、垃圾箱的，可以将配置文件config\default.inc.php中这行的参数改成true
// automatically create the above listed default folders on first login
$config['create_default_folders'] = true;

主题: 关于修改Roundcube Webmail上传附件的大小.
回复总数: 0
阅读次数: 15920

关于修改Roundcube Webmail上传附件的大小.

修改roundCube目录下的.htaccess文件：
php_value upload_max_filesize 30M
php_value post_max_size 30M

修改php.ini 配置文件
upload_max_filesize=30m
post_max_size=30m
重启web服务即可
