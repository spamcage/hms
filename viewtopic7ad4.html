Roundcube 多域名设置方法,按照步骤可完成。

代码: 全选

查找：$rcmail_config['default_host'] =
修改成：
$rcmail_config['default_host'] = array (
    'mail.hmailserver.org' =>   '@hmailserver.org',
        'mail.sina.com' => '@sina.com');

查找：$rcmail_config['username_domain'] =
修改成：
$rcmail_config['username_domain'] = array (
    'mail.hmailserver.org' =>   'hmailserver.org',
        'mail.sina.com' => 'sina.com');

查找：$rcmail_config['mail_domain'] =
修改成：
$rcmail_config['mail_domain'] = $rcmail_config['username_domain'];
